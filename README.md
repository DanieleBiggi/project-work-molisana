# Project Work - La Molisana e la sostenibilità

Questo repository contiene una pagina web sviluppata per il Project Work del corso di laurea in Informatica per le Aziende Digitali.

## Obiettivo

Il progetto ha l'obiettivo di realizzare una pagina HTML e CSS dedicata alla presentazione delle principali iniziative di sostenibilità de La Molisana e al download del Bilancio di Sostenibilità 2024.

## Tecnologie utilizzate

- HTML5
- CSS3
- Responsive design
- Hosting web Aruba
- GitHub per versionamento e condivisione del codice

## Struttura

- `index.html`: struttura della pagina
- `style.css`: stile grafico e responsive
- `assets/report_sostenibilita_molisana_2024.pdf`: documento scaricabile

## Nota

Il progetto è stato realizzato esclusivamente a scopo didattico e non rappresenta un sito ufficiale de La Molisana.
